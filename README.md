 心心念念
=============
##### 这是一个每日推送十个精选商品的购物类软件。后台采用bomb，ui设计由学设计的同学完成，因为精力和水平有限，没有对各类机型进行适配，本人调试机为moto tx1085。大部分功能已经实现，还存在一些小的细节和bug需要完善，有时间了会慢慢更新。
